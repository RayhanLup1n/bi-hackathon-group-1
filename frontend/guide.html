Memulai - Login & Navigasi
Cara Login
1
Buka Halaman Login
Akses aplikasi melalui browser di http://localhost:8000/login (development) atau URL yang diberikan admin untuk production.
2
Masukkan Kredensial
Isi username dan password yang telah diberikan oleh admin. Klik tombol Masuk untuk login.
3
Navigasi Menu
Setelah login, Anda akan diarahkan ke Dashboard. Menu navigasi ada di pojok kanan atas:
Dashboard - Monitoring harga real-time
Panduan - Halaman ini
Prediksi - Prediksi ML (Analyst+)
Admin - Kelola sistem (Admin only)
🔐 Menu yang tampil tergantung role Anda: Viewer hanya bisa akses Dashboard, Analyst bisa akses Prediksi, Admin bisa akses semua halaman.
2
Dashboard - Monitoring Harga
Fitur Utama Dashboard

Dashboard adalah halaman utama untuk memantau harga pangan secara real-time. Di sini Anda bisa melihat status harga, tren, dan alert untuk setiap komoditas.

Menggunakan Filter
1
Filter Komoditas
Pilih komoditas yang ingin dipantau dari dropdown. Tersedia 6 komoditas: Bawang Merah, Bawang Putih, Cabai Merah Besar, Cabai Merah Keriting, Cabai Rawit Hijau, dan Cabai Rawit Merah.
2
Filter Wilayah
Pilih provinsi atau kota untuk melihat data spesifik wilayah. Tersedia 4 provinsi: Banten, Jawa Barat, DKI Jakarta, dan Sulawesi Selatan.
3
Filter Tanggal
Pilih rentang tanggal untuk melihat data historis atau fokus pada periode tertentu. Data tersedia dari tahun 2020 hingga sekarang.
Membaca Grafik & Tabel
📈
Grafik Tren Harga
Menampilkan pergerakan harga dalam periode yang dipilih. Garis merah menunjukkan batas HET (Harga Eceran Tertinggi). Hover pada titik untuk melihat detail harga dan tanggal.
📊
Tabel Harga per Kota
Menampilkan harga terkini di setiap kota pantauan. Kolom status menunjukkan kondisi harga dibanding HET. Klik header kolom untuk mengurutkan data.
🔔
Alert & Notifikasi
Badge berwarna menunjukkan status harga: AMAN (hijau), WASPADA (kuning), KRITIS (merah), MELAMPAUI (merah gelap).
💡 Status harga dihitung berdasarkan perbandingan dengan HET: AMAN (<80%), WASPADA (80-100%), KRITIS (100-120%), MELAMPAUI (>120%).
3
Halaman Prediksi ML
Tentang Prediksi ML

Halaman Prediksi menggunakan model Machine Learning (LightGBM Quantile) untuk memprediksi harga pangan 7-14 hari ke depan. Prediksi dilengkapi dengan analisis FTA (Fault Tree Analysis) dan rekomendasi dari LLM Agent.

Cara Menggunakan
1
Pilih Komoditas & Wilayah
Gunakan filter untuk memilih komoditas dan wilayah yang ingin diprediksi. Sistem akan menampilkan prediksi berdasarkan data historis wilayah tersebut.
2
Lihat Grafik Prediksi
Grafik menampilkan harga historis (garis solid) dan prediksi (garis putus-putus). Area berwarna menunjukkan confidence interval (rentang kemungkinan harga).
3
Baca Analisis FTA
Panel FTA menampilkan faktor-faktor yang mempengaruhi prediksi: hari raya, cuaca ekstrem, persebaran kota, dan kondisi stok.
4
Lihat Rekomendasi LLM
LLM Agent memberikan rekomendasi aksi berdasarkan hasil analisis. Rekomendasi mencakup langkah pencegahan dan mitigasi yang bisa diambil.
Memahami Output Prediksi
📉
Confidence Interval
Area berwarna di sekitar garis prediksi menunjukkan rentang kemungkinan harga. Semakin lebar area, semakin tinggi ketidakpastian prediksi.
🎯
Akurasi Model
Model dilatih dengan data historis 2020-2026. Akurasi bervariasi tergantung komoditas dan kondisi pasar. Gunakan sebagai panduan, bukan kepastian.
🤖
LLM Agent (Groq)
Rekomendasi dihasilkan oleh LLM (Llama 3.3 70B via Groq API). Agent menganalisis hasil FTA dan memberikan saran aksi yang kontekstual.
⚠️ Prediksi ML adalah alat bantu, bukan keputusan final. Selalu pertimbangkan faktor lain yang mungkin tidak tercakup dalam model.
4
Memahami Status & Alert
Status Harga vs HET

Setiap komoditas memiliki HET (Harga Eceran Tertinggi) yang ditetapkan pemerintah. Status harga dihitung berdasarkan perbandingan harga aktual dengan HET.

AMAN Hijau <80% HET
WASPADA Kuning 80-100% HET
KRITIS Merah 100-120% HET
MELAMPAUI Merah Gelap >120% HET
Faktor Analisis FTA

Sistem menganalisis 4 faktor utama yang mempengaruhi harga pangan:

D1
Hari Raya
Apakah tanggal analisis masuk window H-14 s/d H+3 dari hari raya nasional. Lonjakan permintaan menjelang hari besar menyebabkan kenaikan harga.
S1
Cuaca Ekstrem
Apakah ada cuaca ekstrem dalam 7 hari terakhir di wilayah pantauan: hujan >100 mm/hari, kekeringan >14 hari, suhu >38°C, atau angin >60 km/jam.
S2
Persebaran Kota
Apakah kenaikan harga terjadi di banyak kota sekaligus (>60% kota pantauan). Kenaikan serempak menandakan tekanan bersifat nasional, bukan lokal.
S3
Kondisi Stok
Apakah stok pedagang di bawah normal. Stok menipis (<60%) atau kritis (<35%) memperkuat potensi kenaikan harga.
📌 Semakin banyak faktor yang aktif, semakin tinggi risiko kenaikan harga. Gunakan informasi ini untuk mengantisipasi dan merencanakan respons.
5
Halaman Admin (Khusus Admin)
Akses Admin

Halaman Admin hanya bisa diakses oleh user dengan role Admin. Di sini Anda bisa melihat analisis FTA & Bowtie secara real-time dan memantau hasil inference ML.

Fitur Admin
🌳
FTA Dashboard
Visualisasi Fault Tree Analysis yang menampilkan semua ancaman (threat) yang sedang aktif. Setiap ancaman ditampilkan dengan status dan threshold-nya.
🎀
Bowtie Analysis
Menampilkan barrier pencegahan dan mitigasi untuk setiap ancaman. Membantu menentukan respons yang tepat berdasarkan kondisi saat ini.
🤖
ML Inference Real-time
Menampilkan hasil inference dari ML Engine secara real-time. Termasuk prediksi harga, confidence level, dan rekomendasi dari LLM Agent.
📊
System Status
Monitoring status sistem: koneksi database, ML Engine, dan API health. Membantu troubleshooting jika ada masalah teknis.
🔒 Halaman Admin berisi informasi sensitif dan kontrol sistem. Pastikan hanya user yang berwenang yang memiliki akses Admin.
6
Tips & FAQ
Tips Penggunaan
💡
Bookmark Halaman Penting
Bookmark Dashboard dan Prediksi untuk akses cepat. Gunakan filter yang sering dipakai sebagai default.
📱
Akses Mobile
Aplikasi responsive dan bisa diakses dari smartphone. Cocok untuk monitoring cepat saat di lapangan.
🔄
Refresh Data
Data diperbarui setiap hari pukul 07:00 WIB. Refresh halaman untuk melihat data terbaru.
Pertanyaan Umum (FAQ)
?
Mengapa saya tidak bisa akses halaman Prediksi?
Halaman Prediksi hanya bisa diakses oleh user dengan role Analyst atau Admin. Hubungi admin untuk upgrade akses jika diperlukan.
?
Data harga tidak muncul untuk tanggal tertentu?
Data PIHPS tidak tersedia untuk hari libur dan weekend. Coba pilih tanggal hari kerja untuk melihat data lengkap.
?
Prediksi ML tidak akurat?
Prediksi ML adalah estimasi berdasarkan pola historis. Faktor eksternal seperti kebijakan mendadak atau bencana mungkin tidak tercakup dalam model.
?
Bagaimana cara melaporkan bug?
Hubungi tim teknis melalui admin atau email ke tim pengembang. Sertakan screenshot dan langkah untuk mereproduksi masalah.
📞 Butuh bantuan lebih lanjut? Hubungi tim support atau lihat API Documentation untuk integrasi teknis.
